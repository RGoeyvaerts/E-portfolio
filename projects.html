Projects

Robot Soccer

In this project we got the task to make math more interesting using soccer for high school students. Because we are IT students we chose to make robots that are controllable via an app that uses bluetooth.

The math aspect for the students will be programming the robots and calculating the voltages and currents to be able to use the electronics. These robots are programmable in arduino ide and a scratch variant namely pictoblox. In this project i made the whole electrical part of the robot and made the code to be able to use the robot with the app.

Smart meter

In this project we got the task to use a raspberry pi to read the outgoing data of our digital electricity meter and use this data in a dashboard that is easy to use. we also needed to integrate a feature that if your peak power usage is a above a certain value that the circuit breaker of the power circuit in question that is using so much power is turned off. i installed all the software and made the programs to be able to read the data from the meter and see this data in the dashboard.

POP project

In this project we got the task to show your teacher what you had learned in the past 8 months, so i decided to make a key-logger.
This is a malicious program that can run on your computer and register all your key strokes and put these in a file which can be checked by the person who made the program.
My teacher told me that i would be fininshed quite quickly if i only made the key loger so i turned my raspberry pi in to a backup-server as an extra.
